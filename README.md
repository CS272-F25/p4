# (p4) CINE-REWIND: A Movie Rental Website

## Project Description: 

Cine-Reweind is a retro-themeed movie website that is built with HTML, CSS, JavaScript, and Bootstrap. The aim of this project is to recreate a user-friendly, nostalgic website that allows users to browse through a vast collection of classic movies and rent them for a low price.

## Project Deliverables:
Cine-Rewind is a low-cost move rental website focused on classic films, with a rotating selection of recent movies and hidden classical gems. Users can browse the catalog, filter movies by genre, add movies to cart, complete a rental form, and submit movie reviews in a retro-themed user interface.

Our commitment emphasizes 100% customer satisfaction and provides a unique, nostalgic experience for movie lovers.
